Having a component nested inside another component, which estable the parent-child relationship

Different Component of interaction between the parent-child component

  • Input Decorator (@Input): a variable that comes from the parent component to the child component
  • Output Decorator (@Output): defines a variable to be passed to the parent. Or, rather, it defines an EventEmitter that will emit() data to the parent

Intercepting the changes between components

  • Setters and getters
  • ngOnChanges

Template Reference Variables

Access properties of a child properties and methods from the parent component
