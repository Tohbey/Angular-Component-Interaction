Types of binding

  • Interpolation Binding: A one way data binding from the component class to the template

    • {{ interpolation }}

  • Property Binding: A one way data binding from the component class to the property of an hmtl.
  • Event binding (): A one way data binding from the template to the class. use to excute functions
    • {{ count }}
  • Two-way binding: it combine property and event binding together

    • Welcome {{ name }}

Using ViewChild Decorator

ViewChild: Property decorator that configures a view query. The change detector looks for the first element or the directive matching the selector in the view DOM. If the view DOM changes, and a new child matches the selector, the property is updated.
